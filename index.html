PyPLC2SQL

PyPLC2SQL is a OPC based historian for Programmable Logic Controllers (PLCs).

PyPLC2SQL uses web2py's ORM, the web2py_dal, to connect to almost any database. The DAL, along with Barry Barnreiter's OpenOPC module create a great platform for building a data collection server. PyPLC2SQL has many features including varied trigger types, classification of tags and the ability to import/export configurations.

You can download PyPLC2SQL here.

File Backup App

A simple desktop application for backing up files on a scheduled basis.

File Backup App is a wxpython application comprised of three parts: the background service, the frontend gui, and a sqlite database. The frontend connects to the backend utilizing the Pyro4 package. This allows the service to run continuously, backing up files according to a user-defined schedule. The scheduling functionality was created using the 'schedule' module and file syncing via 'dirsync'.

You can download the application source code here.
